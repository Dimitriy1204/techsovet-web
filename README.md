# 🤖 ТехСовет — Веб-интерфейс

Веб-форма для общения с AI-агентом «ТехСовет» через FastAPI-прокси.

## 📌 Описание

Это фронтенд-часть проекта «ТехСовет» — AI-консультанта для технических вопросов.
Интерфейс реализован в виде одностраничного приложения (SPA) на HTML/CSS/JavaScript
и размещён на GitHub Pages.

Проект состоит из трёх компонентов:

1. **Фронтенд** (этот репозиторий) — чат-форма на GitHub Pages
2. **FastAPI-прокси** ([techsovet-api](https://github.com/dimitriy1204/techsovet-api)) — серверная часть, принимающая запросы и обращающаяся к Langflow
3. **Langflow + LLM** — AI-агент «ТехСовет» с инструментами (калькулятор, URL-парсер)

## 🚀 Возможности интерфейса

- ✅ Чат-интерфейс с историей сообщений
- ✅ Крупный, удобный для чтения шрифт
- ✅ Многострочный ввод вопросов (Shift+Enter — новая строка)
- ✅ Выбор режима ответа: краткий или развёрнутый
- ✅ Отображение времени обработки запроса
- ✅ Сохранение настроек в браузере (localStorage)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Индикатор загрузки во время ожидания ответа
- ✅ Обработка ошибок соединения с API

## 🛠️ Технологии

- HTML5
- CSS3 (Tailwind CSS через CDN)
- Vanilla JavaScript (ES6+)
- Fetch API для HTTP-запросов

## ⚙️ Настройка

Перед использованием необходимо указать URL FastAPI-сервиса.

### Способ 1: Через настройки интерфейса (рекомендуется)

1. Откройте сайт на GitHub Pages
2. Нажмите на блок «⚙️ Настройки»
3. Введите URL вашего FastAPI-сервиса в поле «API URL»
4. Настройка сохранится в браузере автоматически

### Способ 2: Изменить в коде

Откройте `index.html` и найдите строку:

```javascript
const CONFIG = {
    apiUrl: 'https://ТВОЙ-САЙТ.up.railway.app',
    ...
};
```

Замените `ТВОЙ-САЙТ.up.railway.app` на реальный URL вашего FastAPI-сервиса.

## 📡 API-контракт

Интерфейс обращается к FastAPI-сервису по эндпоинту `POST /ask`.

### Формат запроса

```json
{
  "message": "Текст вопроса",
  "user_id": "web_user_xxxxx",
  "session_id": "session_1234567890",
  "response_mode": "detailed",
  "source": "web"
}
```

### Формат ответа

```json
{
  "input": "Текст вопроса",
  "user_id": "web_user_xxxxx",
  "session_id": "session_1234567890",
  "auth_used": "bearer",
  "result_text": "Ответ ТехСовета...",
  "response_mode": "detailed",
  "process_time_seconds": 3.4521,
  "source": "web",
  "raw": { ... }
}
```

## 📋 Требования к FastAPI-серверу

Для корректной работы необходимо:

1. FastAPI-сервис должен быть доступен по HTTPS
2. На сервере должна быть настроена CORS-политика, разрешающая запросы с домена GitHub Pages:
   ```
   GITHUB_PAGES_ORIGIN=https://dimitriy1204.github.io
   ```

## 🎮 Использование

1. Откройте сайт: https://dimitriy1204.github.io/techsovet-web/
2. Убедитесь, что в настройках указан корректный URL API
3. Введите технический вопрос в поле ввода

- **Enter** — отправить сообщение

## 🤖 О проекте «ТехСовет»

ТехСовет — это AI-агент, реализованный в виде коллектива экспертов:
конструктор, физик-теоретик, опытный механик, технолог, начальник производства,
электронщик, электрик, термист и экономист.

Агент работает на базе Langflow и использует:
- **Калькулятор** — для точных математических расчётов
- **URL-парсер** — для получения информации с веб-страниц
- **LLM-модель** — для генерации ответов (локальная Ollama или облачная YandexGPT)

## 🐛 Известные ограничения

- Интерфейс не работает, если FastAPI-сервис недоступен
- Для работы требуется интернет-соединение
- Настройки сохраняются только в текущем браузере
- При использовании локального Langflow необходим туннель (ngrok/cloudflared)
