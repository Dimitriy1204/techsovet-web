# 🤖 ТехСовет — Веб-интерфейс

Веб-форма для общения с AI-агентом "ТехСовет" через FastAPI-прокси.

## 📌 Описание

Это фронтенд-часть проекта "ТехСовет" — AI-консультанта для технических вопросов. 
Интерфейс реализован в виде одностраничного приложения (SPA) на HTML/CSS/JavaScript 
и размещён на GitHub Pages.

Проект состоит из трёх компонентов:

1. **Фронтенд** (этот репозиторий) — чат-форма на GitHub Pages
2. **FastAPI-прокси** — серверная часть, принимающая запросы и обращающаяся к Langflow
3. **Langflow + LLM** — AI-агент "ТехСовет" с инструментами (калькулятор, URL-парсер)

## 🚀 Возможности интерфейса

- ✅ Чат-интерфейс с историей сообщений
- ✅ Крупный, удобный для чтения шрифт
- ✅ Многострочный ввод вопросов (Shift+Enter — новая строка)
- ✅ Выбор режима ответа: краткий или развёрнутый
- ✅ Отображение времени обработки запроса
- ✅ Сохранение настроек в браузере (localStorage)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Индикатор загрузки во время ожидания ответа
- ✅ Обработка ошибок соединения с API

## 🛠️ Технологии

- HTML5
- CSS3 (Tailwind CSS через CDN)
- Vanilla JavaScript (ES6+)
- Fetch API для HTTP-запросов
- GitHub Pages для хостинга
